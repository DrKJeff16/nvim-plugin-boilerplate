# nvim-plugin-template

An extended Neovim Lua plugin template with `pre-commit`, StyLua, selene,
and GitHub actions included from the start.

To configure the template simply run `generate.sh` in your terminal:

```bash
./generate.sh # Has to be run in the repository root!
```
